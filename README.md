# ✍🏻 Como instalar

Clone o projeto, utilizando o terminal:

```bash
  git clone https://github.com/mariquegonn-dev/portfolio-next-14
```

Em seguida:

```bash
  npm install
```

Para rodar o projeto:

```bash
  npm run dev
```
